# Cyclistic Bike-Share Analysis



Project Overview
This project analyzes Cyclistic bike-share data to identify differences between casual riders and members, with the goal to increase membership conversions.



Business problem
How do casual riders and members use Cyclistic bikes differently?



Tools used
* Python (pandas)
* Tableau Dashboard
* Jupyter Notebook



Key Insights
* Casual riders take longer trips on average
* Members ride more frequently, especially on weekdays and during commuting hours
* Casual riders are more active on weekends
